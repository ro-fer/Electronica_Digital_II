# Electrónica Digital II

Repositorio con ejercicios y prácticas desarrolladas para la materia **Electrónica Digital II**, enfocado en programación en lenguaje C y desarrollo sobre microcontroladores STM32.

## 📚 Contenido

El repositorio incluye prácticas sobre:

- 🖥️ Programación en lenguaje C.
- 🔵 Uso de la placa STM32 Blue Pill.
- 📡 Comunicación UART.
- ⚡ Transferencia de datos mediante DMA.
- 🛠 Programación a bajo nivel.

---

## 🔵 Práctica Blue Pill

Conjunto de ejercicios realizados sobre la placa **STM32F103C8T6 (Blue Pill)**.

Temas abordados:

- Configuración de GPIO.
- Manejo de LEDs y pulsadores.
- Acceso a registros.
- Programación bare metal.
- Uso de periféricos integrados.

---

## 📡 Práctica UART

Implementación y pruebas de comunicación serial UART.

Incluye:

- Configuración del periférico UART.
- Transmisión y recepción de datos.
- Comunicación con terminales seriales.
- Ejemplos prácticos de envío y recepción.

---

## ⚡ Práctica DMA

Ejercicios relacionados con el controlador **DMA (Direct Memory Access)**.

Se estudia:

- Configuración de canales DMA.
- Transferencia de datos sin intervención de la CPU.
- Optimización del rendimiento.

---

## 🖥️ Práctica de Lenguaje C

Ejercicios introductorios y avanzados en C:

- Variables y tipos de datos.
- Estructuras de control.
- Funciones.
- Punteros.
- Arreglos y matrices.
- Manipulación de memoria.

---

## 🛠 Herramientas utilizadas

- Lenguaje C
- STM32 Blue Pill (STM32F103C8T6)
- GCC ARM Embedded
- ST-Link
- Makefile
- Git & GitHub
